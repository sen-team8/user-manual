# Adding a task to the List


---



User can add any number of tasks in the list. <br/>

* Enter the task in the textbox.<br/><br/>
![](todoadd.png)


* Click on the `Plus` button placed on the right-top corner.<br/><br/> 
![](todoadd2.png)


* This will add task to the list.
<br/><br/>
